## 데이터 관련

- 왜 이 데이터를 선택했는가?

  → [취업 준비생이 실제 채용 공고를 기반으로 필요한 역량과 기술을 분석할 수 있도록 하기 위해 취업 공고 데이터를 선택했습니다. 사용자의 보유 역량과 공고에서 요구하는 역량을 비교하여 맞춤형 추천을 제공하는 것이 프로젝트의 목표입니다.]

- SQLite와 ChromaDB에 같은 데이터를 저장하는 이유는 무엇인가?

  → [SQLite는 공고 정보를 구조화된 형태로 저장하고 조건 검색이나 조회에 활용하기 위해 사용했습니다. ChromaDB는 문서의 의미를 기반으로 유사한 공고를 검색하는 RAG 기능을 위해 사용했습니다. 두 데이터베이스를 함께 사용하여 정확한 조회와 의미 기반 검색을 모두 지원할 수 있도록 했습니다.]
